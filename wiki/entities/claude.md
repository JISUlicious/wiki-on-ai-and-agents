# Claude

Claude is the family of [[decoder-only-transformer|decoder-only Transformer]] LLMs produced by [[anthropic|Anthropic]]. Named after Claude Shannon. The Claude family has gone through several major generations (each with multiple size tiers — typically Haiku / Sonnet / Opus, smallest to largest):

| Generation | Released |
|---|---|
| Claude 1 | March 2023 |
| Claude 2 | July 2023 |
| Claude 2.1 | November 2023 |
| Claude 3 (Haiku / Sonnet / Opus) | March 2024 |
| Claude 3.5 Sonnet | June 2024; updated October 2024 |
| Claude 3.5 Haiku | October 2024 |
| Claude 3.7 Sonnet | February 2025 |
| Claude 4 (Sonnet / Opus) | mid-2025 |
| Claude Sonnet 4.5 | early 2026 (studied in [[emotion-concepts-anthropic-2026]]) |
| Claude 4.6 Sonnet, Claude 4.7 Opus, Haiku 4.5 | 2025–2026 |

Architectural details are not officially disclosed. Like [[gpt-4]], Claude is a closed-weights model accessed via the [[anthropic|Anthropic]] API or the **Claude.ai** product.

## Notable capabilities

- Strong on coding and reasoning (powers Claude Code, Cursor, and many other coding tools).
- Native tool use / function calling via a structured API.
- Long context (up to 200k tokens in the 3.x generation; 1M-token variants from 4.x).
- Multimodal (image input in 3.x onwards).
- Constitutional AI training — preference data partially generated by self-critique against a written set of principles.
