# Mario Zechner ("badlogic")

Austrian developer; longtime gamedev (libGDX creator) turned LLM-agent author. Creator and primary maintainer of [[pi-mono|pi]], a minimalist terminal coding agent distinguished by editorial restraint — explicitly refusing sub-agents, plan mode, MCP, and other now-common features in its core.

His public stance, repeated across the pi-mono / pi-skills documentation: build the smallest possible useful core; push everything else into extensions; respect the user's ability to compose features.
