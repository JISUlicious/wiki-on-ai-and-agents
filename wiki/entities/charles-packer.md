# Charles Packer

UC Berkeley PhD student and co-founder of [[letta]], the company productizing MemGPT. First author of MemGPT (2023) and Sleep-time Compute (2025), both projects exploring memory and inference-time computation patterns for LLM agents.

## Notable work
- [[memgpt-packer-2023]] — first author; introduced [[memgpt]] and the OS-inspired memory hierarchy for LLM agents.
- [[sleep-time-compute-lin-2025]] — first author; introduced [[sleep-time-compute]] as a paradigm for offline agent reasoning.
