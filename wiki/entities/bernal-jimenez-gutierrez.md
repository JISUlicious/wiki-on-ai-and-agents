# Bernal Jimenez Gutierrez

PhD student at [[ohio-state-university]] advised by Yu Su. Works on retrieval-augmented generation and memory systems for LLMs, with a focus on hippocampus-inspired architectures that bind episodic recall, schema, and associative retrieval into a single non-parametric memory layer for language agents.

First author of HippoRAG and HippoRAG 2, which introduced a personalized-PageRank-over-knowledge-graph retrieval pattern aimed at multi-hop, sense-making, and associative-memory tasks that flat dense retrievers handle poorly.
