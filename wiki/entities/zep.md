# Zep

Company building long-term-memory infrastructure for LLM agents. Ships Zep Cloud, a managed memory service, and Graphiti, a temporal knowledge-graph engine that incrementally extracts entities and relations from agent conversations and tool traces.

Frequently cited as a baseline in 2025 agent-memory papers — comparison rows for Zep appear in evaluations from competing systems like [[mem0]] and [[mirix]], making it a de facto reference implementation for graph-structured agent memory.

## Notable work

- Graphiti — temporal knowledge-graph memory engine (open source).
- Zep Cloud — hosted long-term memory API for agents.
