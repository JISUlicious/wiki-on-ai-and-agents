# OpenHands

OpenHands (formerly OpenDevin) is an open-source platform for building generalist software-development agents. It provides a common reference harness — execution sandbox, tools, and event stream — that researchers and practitioners reuse to run and compare agents. It is a widely cited instance of [[code-as-harness]] infrastructure.
