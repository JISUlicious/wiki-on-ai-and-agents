# Long-Term Memory

Long-term memory refers to information an agent persists across sessions, as opposed to short-term context held only within a single conversation window. It is the substrate that [[memory-management]] curates — deciding what to store, retrieve, update, or forget over time. Robust long-term memory is a prerequisite for agents that accumulate knowledge and improve with experience.
