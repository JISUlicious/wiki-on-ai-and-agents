# ChatDev

ChatDev is a communicative multi-agent framework for software development that organizes LLM agents into waterfall-style roles (design, coding, testing, documentation) operating over a shared codebase. Agents collaborate through structured natural-language dialogue to decompose and complete the development pipeline. It is an early example of [[code-as-harness]] multi-agent systems.
