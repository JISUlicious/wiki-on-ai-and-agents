# AgentCoder

AgentCoder is a multi-agent code-generation framework built from a triad of specialized agents: a Programmer, a Test-Designer, and a Test-Executor. The agents iterate together — generating code, designing tests, and running them — so feedback from execution drives refinement. It is an instance of [[code-as-harness]] applied to program synthesis.
