# Memory Consolidation

Memory consolidation is the process of reprocessing and compacting accumulated memories during idle time into a more durable, retrievable form. By summarizing, deduplicating, and reorganizing raw experience offline, an agent keeps its memory store compact and useful. It is closely tied to [[memory-management]], [[offline-consolidation]], and [[sleep-time-compute]].
