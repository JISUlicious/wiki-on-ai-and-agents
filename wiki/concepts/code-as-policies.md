# Code as Policies

Code as Policies is a code-for-acting approach for embodied and robotic settings: the LLM emits Python that orchestrates perception and control APIs to carry out a task. The generated programs can compose primitives, use loops and conditionals, and call into the robot's skill library. It extends the [[code-as-harness]] idea from reasoning into physical action.
