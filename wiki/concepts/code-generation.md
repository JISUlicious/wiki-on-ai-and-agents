# Code Generation

Code generation is the task of producing source code from natural-language specifications or partial context. It ranges from completing single functions to resolving whole repository-level issues, and is commonly evaluated on benchmarks such as HumanEval and [[swe-bench]]. Agentic systems increasingly treat code generation as an iterative loop within a [[code-as-harness]] environment.
